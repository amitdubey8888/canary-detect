# Canary Detect


## Live URL: https://canary-detect.herokuapp.com


## Quick Start

  Clone the repository:

```bash
$ git clone https://github.com/amitdubey8888/canary-detect.git
```

  Goto project directory:

```bash
$ cd canary-detect
```

  Install dependencies:

```bash
$ npm install
```

  Start the app:

```bash
$ npm start
```

  View the app at: http://localhost:3000
